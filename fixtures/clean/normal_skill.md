# Code Formatter Skill

This skill helps format Python code according to PEP 8 guidelines.

## Usage

Provide a Python file and this skill will:
1. Fix indentation to use 4 spaces
2. Add appropriate blank lines between functions
3. Organize imports alphabetically
4. Ensure consistent string quoting

## Examples

```python
def hello_world():
    print("Hello, World!")
```

## Notes

- This skill only modifies whitespace and formatting
- It does not change program logic
- Compatible with Python 3.8+
